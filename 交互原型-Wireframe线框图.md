## 3. 交互规范

### 3.1 手势交互

| 场景 | 手势 | 行为 |
|------|------|------|
| 列表页 | 下拉 | Pull-to-Refresh 刷新数据 |
| 时间轴 | 左右滑动 | 切换日期 |
| 任务卡片 | 右滑 | 快捷完成打卡 |
| 任务卡片 | 左滑 | 显示编辑/删除选项 |
| 图表 | 捏合 | 缩放时间范围 |
| 图表 | 长按 | 显示数据点详情 |
| 弹窗 | 下滑 | 关闭弹窗 |

### 3.2 过渡动画

| 场景 | 动画类型 | 时长 |
|------|----------|------|
| 页面切换 | Push/Pop | 350ms |
| 弹窗出现 | Bottom Sheet Slide | 300ms |
| 打卡成功 | Scale + Confetti | 500ms |
| 数据加载 | Skeleton Shimmer | - |
| 图表绘制 | Line Draw | 800ms |
| 环形进度 | Arc Animate | 600ms |

### 3.3 反馈机制

| 操作 | 反馈类型 |
|------|----------|
| 点击按钮 | Haptic Feedback (Light) |
| 打卡成功 | Haptic (Success) + 动画 + Toast |
| 表单错误 | Shake + 红色边框 + 提示文字 |
| 网络错误 | Alert + 重试按钮 |
| 加载中 | Spinner / Skeleton |
| 空状态 | 插图 + 引导文案 + 操作按钮 |
